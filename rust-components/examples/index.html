Code Example

Here's how to use the NeuraDeSci Rust components from JavaScript:

import * as neuradesci from 'neuradesci-core';

// Create EEG data
const eegData = await neuradesci.create_eeg_data(
  256.0, // sampling rate in Hz
  "patient-123", 
  "researcher-001"
);

// Encrypt sensitive data
const keys = neuradesci.generate_keys();
const encryptedData = await neuradesci.encrypt_data(
  JSON.stringify(eegData), 
  keys.privateKey
);

// Store on IPFS
const ipfsResult = await neuradesci.upload_to_ipfs(
  encryptedData,
  "eeg_study.json"
);

// Analyze the data
const analysisResults = await neuradesci.analyze_eeg_data(
  JSON.stringify(eegData)
);
